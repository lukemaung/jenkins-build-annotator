Enable Build Annotator Plugin to provide programmatic CRUD access to build annotations.

Build Annotations are arbitrary blocks of text in key-value pair format that can be used to annotate builds. Annotations are displayed on the main Build Page.

If ENABLED, build annotation CRUD is enabled on new builds.

If DISABLED, build annotation CRUD is disabled on new builds.

NOTE: One CANNOT add build annotations to builds that ran in the past when the plugin was disabled or not installed.
